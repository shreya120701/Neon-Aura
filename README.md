# Neon Aura

Neon Aura is an interactive system that combines dynamic neon lighting effects and audio feedback, controlled through intuitive hand gestures. It enables seamless, touch-free interaction with commonly used web applications.

## Features

* 🎇 Real-time neon aura visual effects
* 🔊 Responsive sound feedback
* ✋ Gesture-based controls for hands-free interaction
* 🌐 Instant access to frequently used web platforms

## Gesture Controls

Use the following hand gestures to trigger specific actions:

| Gesture         | Description                                                     | Action                  |
| --------------- | --------------------------------------------------------------- | ----------------------- |
| ☝️ Index Finger | Extend only the index finger while keeping other fingers curled | Opens **ChatGPT**       |
| ✊ Closed Fist  | Make a full fist with your hand                                 | Opens **WhatsApp Web**  |
| ✌️ Peace Sign   | Extend index and middle fingers to form a “V” shape             | Triggers **Live Heart** |

## How It Works

The system uses real-time hand gesture recognition to map predefined gestures to corresponding actions. Each gesture activates a unique combination of neon lighting and sound effects, creating an engaging and immersive user experience.
